R-Type Effect

Shooter Sci-fi 3D Pixel Art Unity VR

Project introduction

R-Type Effect is a VR shooter game inspired by the classic R-Type series. The player must fight through waves of enemies to reach the end of the level. The game features a dynamic AI system that adapts to the player's actions.

My Role

In R-Type Effect, I took on two key roles:
Scrum Master : I facilitated team collaboration, ensuring smooth workflow and efficient task management. I helped coordinate development sprints, track progress, and resolve any roadblocks to keep the project on schedule.
Gameplay Programmer : I was responsible for implementing core gameplay mechanics, focusing on player controls, enemy behaviors, and combat interactions. My work ensured responsive gameplay and a faithful adaptation of the R-Type experience.

Gameplay mechanics

Simon Says :
Le joueur doit reproduire une séquence de couleurs pour valider la séquence. La partie est constituer de 5 séquences générée aléatoirement et recommence de zéro en cas d'échec d'une séquence.

Tools and Technologies

Unity : Main game engine, used to integrate VR features and create gameplay interactions.
C# : For programming the mechanics and interactions.
Oculus Quest 2 : VR headset used as the main platform for the game, providing an immersive virtual reality experience.
Discord : Team communication to coordinate the technical and creative aspects of the project.
Github : Version control platform used to manage and collaborate on the project's codebase, allowing team members to track changes, fix issues, and ensure code stability throughout development.
