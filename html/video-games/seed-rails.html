SeedRails

Action Strategy 3D Global Game Jam Singleplayer Unity

Project introduction

For the Global Game Jam 2023, the theme was "roots". Seed Rail is a prototype developed in two days.

My Role

In Seed Rail, my primary focus was on developing the enemy AI system. This included implementing behavior patterns and designing a dynamic shooting system to enhance challenge and gameplay depth.

Gameplay mechanics

Enemy AI System :
The enemies in Seed Rail are equipped with dynamic AI that reacts to the player’s actions and navigates the environment intelligently. Their objective is to target and destroy the player’s repeaters one by one, slowing them down until the last repeater is eliminated.

Projectile System :
Enemies feature an automated shooting system that exclusively targets the repeaters instead of the player, increasing tension and urgency in gameplay. Conversely, the player benefits from an auto-shooting system that prioritizes the closest enemy, encouraging fast-paced and strategic play.

Enemy Spawn System :
An enemy spawner adds an element of surprise by randomly generating a number of enemies within a predefined range, bringing variety and dynamism to the gameplay.

Tools and Technologies

Unity : Main game engine, used to integrate features and create gameplay interactions.
C# : For programming the mechanics of enemy AI system.
Discord : Team communication to coordinate the technical and creative aspects of the project.
Github : Version control platform used to manage and collaborate on the project's codebase, allowing team members to track changes, fix issues, and ensure code stability throughout development.
