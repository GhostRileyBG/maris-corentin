Gameplay mechanics

Simon Says :

Tools and Technologies

Unity : Main game engine, used to integrate VR features and create gameplay interactions.
C# : For programming the mechanics of puzzles and interactions.
Oculus Quest 2 : VR headset used as the main platform for the game, providing an immersive virtual reality experience.
Discord : Team communication to coordinate the technical and creative aspects of the project.
Github : Version control platform used to manage and collaborate on the project's codebase, allowing team members to track changes, fix issues, and ensure code stability throughout development.
